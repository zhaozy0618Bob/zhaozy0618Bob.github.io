## 你好，我是传奇绳匠

一个游走在空洞与现实之间的情报商人。白天写代码，深夜刷绳网。

## 这个博客

**HollowLight** 是我用来记录一切的地方：

- 技术笔记与踩坑经验
- 游戏随想与角色分析  
- 深夜城市漫游日记
- 偶尔的摄影与随笔

博客主题同样叫 HollowLight，是一个深度致敬《绝区零》绳网界面的 Hexo 主题，[在 GitHub 开源](https://github.com)。
