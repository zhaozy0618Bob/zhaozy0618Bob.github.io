这是 **HollowLight** —— 一个深度致敬《绝区零》「绳网」论坛 UI 的 Hexo 主题。纯黑背景、荧光黄强调色、标志性右下角切角，让你的博客像赛博朋克世界里的情报中转站。

<!-- more -->

## 设计灵感

《绝区零》游戏内的「绳网」是邦布们在空洞中传递情报的暗网平台，其界面设计充满了赛博朋克质感：

- **纯黑底色** — `#000000`，无妥协的深邃
- **荧光黄强调** — `#fbfe00`，来自 ZZZ 的标志性色彩
- **切角卡片** — 右下角 `border-radius: 0`，这是 ZZZ UI 的识别符
- **斜切 Tab** — 导航选中态使用 `skew(-27deg)` 平行四边形背景

## 快速开始

```bash
# 克隆主题
git clone https://github.com/your-username/HollowLight themes/HollowLight

# 在站点 _config.yml 中设置
theme: HollowLight
```

点击任意文章标题，体验 **iframe 弹窗阅读模式** — 页面不刷新，沉浸感满分。
